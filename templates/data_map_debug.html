How to use this API

From the home route, use one of our available API routes to search for US county demography data by state or year.


Required variable formats:

  • (year): Any year between and including 2009-2018
  • (state): The name of any 50 US states, beginning with capital letters. Does not include US territories or the District of Colombia

Available Routes

  • /by_year/(year)
  • /by_state_name/(state)
  • /by_state_year/(state)/(year)
